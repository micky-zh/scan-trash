# Screening Workflow

## Stage 1: Universe

目标：拿到完整、可去重、可更新的港股股票池。

优先级：

1. HKEX 官方证券列表
2. AKShare 行情与财务接口
3. 本地统一代码表

## Stage 2: First Pass Filter

先删掉：

- 仙股
- 长期亏损股
- 极低流动性
- 明显杠杆过高
- 无法解释业务模式的公司

## Stage 3: Alphabetical Review

按字母或拼音首字母顺序浏览：

- 公司名
- 代码
- 价格
- 市值
- PE / PB
- ROE / ROIC
- 股息率

发现异常便宜、但经营质量不差的公司后，进入下一轮。

## Stage 4: Deep Dive

至少补齐：

- 业务模式
- 竞争优势
- 管理层资本配置
- 现金流质量
- 负债结构
- 周期敏感性
- 历史分红与回购
