# Investment Principles

## Purpose

先建立一个高信噪比的股票观察池，再进入深入分析。

## First Principles

1. 便宜不是理由，理解才是理由。
2. 第一轮筛选的任务是排除，不是证明值得买。
3. 规则必须允许例外，但例外必须有记录。
4. 任何规则如果连续制造坏样本，就必须重写。
5. 研究日志比一次性的灵感更重要。

## What This System Should Avoid

- 因为低 PE 就误判周期股
- 因为低 PB 就忽视资产质量
- 因为股息高就忽视分红不可持续
- 因为名字陌生就跳过潜在好公司
- 因为名字熟悉就放松质量要求
