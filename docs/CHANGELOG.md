# 变更记录

记录规则：

- 最新版本放在最上面。
- 每次更新都记录新增、删除、变更和时间。
- 如果是破坏性变更，优先写清影响范围和替代用法。

## v0.1.0 - 2026-05-05

新增：

- 支持美股 `filings --market us` 缓存 SEC 申报索引。
- 支持美股 `filings --download` 仅保存索引和主文档。
- 支持 `financials --missing-only` 按文件补缺。
- 支持 `filings --workers` 并发跑公告任务。
- 新增 README 的快速上手、命令区别、工作流和常见问题说明。

变更：

- `financials` 默认仍然按股票增量追加，但 `--missing-only` 现在只补缺失的 statement 文件。
- `filings` 命令现在支持 `--workers`，默认保持保守串行。
- README 的术语统一为“公告索引 + 主文档”。
- 美股 filings 从“主文档 + 附件目录”收紧为“索引 + 主文档”。
- 财报和公告缓存默认只保留最近 5 年。

删除：

- 删除了旧的混杂式 README 叙述，改为命令导向的结构说明。
- 删除了美股 filings 的附件批量下载策略，只保留主文档下载。

备注：

- 当前版本记录从 `v0.1.0` 开始，后续每次更新请在顶部追加新版本。
