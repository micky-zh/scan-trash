# Review Loop

## Thesis Log

For every company, maintain a `thesis-log.md` with:

- Original thesis.
- Core assumptions.
- Evidence.
- Disconfirming signals.
- Monitoring indicators.
- Updates by date.

## Tracking Review

For quarterly or annual review, compare new facts against the original thesis:

- Which assumptions are stronger?
- Which assumptions are weaker?
- Which assumptions failed?
- Did valuation change because business value changed or because price changed?
- Is the action still rational?

## Sell Criteria

Check explicitly:

1. Price severely exceeds conservative intrinsic value.
2. Moat is structurally impaired.
3. Management integrity issue appears.
4. A significantly better opportunity exists.
5. Original thesis is disproved by facts.

Management integrity problems require immediate escalation and usually override valuation.

## Framework Updates

When a lesson changes the framework, update:

- `investment-research/framework/99-待完善框架.md`
- `investment-research/logs/framework-changelog.md`
- `investment-research/TODO.md`

Record:

- What changed.
- Why it changed.
- What evidence caused the change.
- How to verify whether the new rule improves decisions.
