# 投资原则

## 目标

买好公司，以合理或便宜价格买入，用正确方式持有，并在事实变化时承认错误。

## 分析顺序

```text
生意质量 -> 管理层与治理 -> 财务安全 -> 估值与安全边际 -> 跟踪与复盘
```

财务分析顺序：

```text
资产负债表 -> 现金流量表 -> 利润表 -> 三表勾稽 -> 估值
```

## 投资前必须回答

1. 这家公司如何赚钱？一句话能否说清楚。
2. 这是不是一个长期存在且不容易被替代的生意？
3. 我认为市场错在哪里？
4. 主要风险是什么？
5. 为什么是我能看懂这个机会？
6. 如果三年后证明我错了，最可能错在哪里？
7. 什么信号出现时必须停止持有？

## 行为纪律

- 市场先生是仆人，不是导师。
- 不因价格下跌自动补仓，只因事实和赔率更好才加仓。
- 不用短期股价验证长期判断。
- 不把低估值等同于安全边际。
- 不买无法解释现金流来源的利润。
- 管理层诚信问题是硬性否决项。

## 仓位约束

- 单一标的原则上不超过可投资资产的 20%。
- 单一行业原则上不超过可投资资产的 40%。
- 高不确定性标的必须降低仓位，不能用仓位弥补认知不足。
- 买入前必须写出最大可承受亏损和触发复盘条件。
